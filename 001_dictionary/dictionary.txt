Android Native Development Kit (NDK) : 	The Android Native Development Kit (NDK) provides a cross-compiling tool
							for compiling code written in C/C++ can be compiled to ARM,
							or x86 native code (or their 64-bit variants) for Android.[3][4] 
							The NDK uses the Clang compiler to compile C/C++

Android version : 				corresponds with target API level, for example Android 13 corresponds with API level 33
